# 🌈 重力小精灵机器人 (Gravity Spirit Bot) v3.0

一个为孩子们设计的、充满魔力的星际重力实验游乐场。

## 🌟 项目简介 (Introduction)
“重力小精灵”是一款基于 Web 的互动物理教学 AUI（智能交互界面）。它通过生动可爱的视觉效果和趣怪的音效，带领孩子们探索地球、月球和木星之间截然不同的重力环境，让枯燥的物理常识变得生动有趣。

## 🎨 核心特性 (Key Features)
- **儿童友好设计**：采用“糖果太空”视觉风格，大字体、圆润形状及丰富的 Emoji 交互。
- **动态物理仿真**：实时模拟不同星球的重力加速度（G），支持调整物体质量（KG）。
- **多媒体沉浸感**：
  - 自动播放的欢乐背景音乐。
  - 专属月亮探险音频指南（月球蹦跳指南.mp3）。
  - Bouncy（跳跃）、Pop（点击）、Achievement（解锁）等趣怪音效。
- **科学探索墙**：记录每一次探险高度，随机播报有趣的重力冷知识。

## 🚀 快速启动 (Quick Start)
1. 下载本项目文件夹。
2. 双击 `index.html` 即可在浏览器中开启探索。
3. 点击 **“进入实验室 ✨”** 按钮以解锁音频权限并激活重力引擎。

## 📁 目录结构 (Structure)
- `index.html`: 实验室骨架。
- `style.css`: “糖果太空”视觉系统。
- `logic_core.js`: 物理引擎、音频管理及交互逻辑。
- `*.png`: 自定义生成的星球背景大图。
- `*.mp3`: 趣怪音效与指南音频。
